에듀플로는 어떤 플랫폼인가요? — 첫 만남 (1차시)

1차시 (50분) · 1부 에듀플로 시작하기

🎯 이 장에서 배우는 것

섹션 제목: “🎯 이 장에서 배우는 것”
  • 에듀플로가 무엇을 만들어 주는 도구인지 한 문장으로 설명할 수 있다
  • 교사용 교재 생성 5단계(Step 1-5)의 흐름을 이해한다
  • ChatGPT 같은 범용 AI와 에듀플로의 차이를 구별할 수 있다

💬 왜 이걸 배우나요?

섹션 제목: “💬 왜 이걸 배우나요?”

선생님께 먼저 한 가지 여쭤볼게요.

“교재 한 챕터를 만드는 데, 보통 몇 시간 걸리셨나요?”

아마 2-3시간은 기본일 겁니다. 자료 찾고, 학습목표 쓰고, 예시 짜고, 활동지 만들고, PDF로 뽑고, 학생들에게 공유까지. 반나절이 훌쩍 갑니다.

에듀플로는 이 과정을 “대화 5번으로 교재 사이트 한 권” 으로 줄여주는 도구입니다. 지금 선생님이 읽고 계신 이 자료도 에듀플로로 만들어졌습니다. 연수가 끝날 무렵, 선생님도 본인 교과의 교재 한 권을 만들어 가시게 됩니다.

이 차시의 목표는 단순합니다. “에듀플로가 뭐 하는 도구인지” 감을 잡는 것. 실습은 2차시부터 시작합니다. 오늘은 편하게 둘러보세요.

📚 핵심 개념

섹션 제목: “📚 핵심 개념”

개념 1: 에듀플로는 “교재 공장”입니다

섹션 제목: “개념 1: 에듀플로는 “교재 공장”입니다”

에듀플로는 마치 자동화된 제빵소와 같습니다. 선생님이 “초코 식빵, 2kg, 당도 낮게”라고 주문하면, 반죽부터 굽기까지 알아서 해서 완제품 빵을 내주는 곳. 선생님은 재료를 직접 계량하지도, 오븐 온도를 맞추지도 않습니다. “어떤 빵을 원하는지”만 말하면 됩니다.

정의하자면:

에듀플로는 선생님이 교재의 방향과 대상을 대화로 알려주면, AI가 목차 → 각 챕터 본문 → 배포 가능한 웹사이트까지 자동으로 만들어 주는 교육자료 생성 플랫폼입니다.

결과물은 PDF 한 장이 아니라, 학생들이 링크로 접속해서 읽는 교재 사이트입니다. 체크박스, 접기/펼치기 정답, 다이어그램이 살아있는 형태로요.

개념 2: 5단계 흐름 — Step 1에서 Step 5까지

섹션 제목: “개념 2: 5단계 흐름 — Step 1에서 Step 5까지”

에듀플로의 전체 과정은 5단계로 정해져 있습니다. 선생님이 외울 필요는 없지만, 흐름을 알면 “지금 내가 어디쯤 와 있는지” 감이 잡힙니다.

각 단계가 만들어 내는 결과물은 다음과 같습니다.

단계하는 일결과물
Step 1. 방향 논의”누구에게, 왜, 어떤 톤으로?”를 대화로 정함교재 기획서 초안
Step 2. 목차 제안AI가 차시별 목차를 자동 생성챕터 리스트
Step 3. 목차 확정선생님이 수정·삭제·추가최종 목차
Step 4. 챕터 생성각 챕터 본문을 한 번에 작성완성된 교재 원고
Step 5. 배포웹사이트로 게시, 링크 공유학생 접속용 URL

핵심은 Step 1-3은 “설계”, Step 4는 “제작”, Step 5는 “납품” 이라는 점입니다. 설계에 시간을 들일수록 제작 결과물이 좋아집니다.

🆚 ChatGPT랑 뭐가 달라요?

섹션 제목: “🆚 ChatGPT랑 뭐가 달라요?”

가장 많이 받는 질문입니다. “그냥 ChatGPT에 시키면 되는 거 아닌가요?”

결론부터 말씀드리면 — 할 수는 있지만, 현실에서는 잘 안 됩니다. 이유는 세 가지입니다.

범용 AI로 교재 만들 때의 한계 3가지

섹션 제목: “범용 AI로 교재 만들 때의 한계 3가지”

① 기억 소실 ChatGPT로 10개 챕터짜리 교재를 만든다고 해봅시다. 1-3장까지는 잘 씁니다. 그런데 7장쯤 가면 “앞에서 정의한 용어”를 잊어버립니다. 1장에서 ‘노드’라고 부르던 걸 7장에서 ‘블록’이라고 부르는 식입니다.

② 템플릿 불일치 1장은 학습목표·예시·정리 순서인데, 5장은 갑자기 정리부터 나옵니다. 매 챕터 “똑같은 형식으로 써줘”라고 지시해도, 미묘하게 어긋납니다. 결국 선생님이 일일이 손봐야 합니다.

③ 배포 공백 ChatGPT는 텍스트만 뱉습니다. 학생에게 공유하려면 — 워드에 붙여넣기 → 서식 정리 → PDF로 내보내기 → 학급 클래스룸 업로드. 이 과정이 챕터마다 반복됩니다.

에듀플로는 어떻게 해결하나요?

섹션 제목: “에듀플로는 어떻게 해결하나요?”
문제ChatGPT에듀플로
기억 소실대화가 길어지면 앞 내용을 잊음교재 전체 컨텍스트를 고정·공유
템플릿 불일치매번 다른 형식으로 출력챕터 템플릿이 강제 적용됨
배포 공백텍스트만 → 수동 가공 필요웹사이트 URL로 즉시 배포

간단히 말하면, ChatGPT가 “만능 일꾼” 이라면 에듀플로는 “교재 전문 공장” 입니다. 할 줄 아는 건 적지만, 그 한 가지를 아주 잘합니다.

섹션 제목: “🔨 따라하기 / 활동”

섹션 제목: “Step 1: 에듀플로 교재 사이트 둘러보기 (15분)”

지금 읽고 계신 이 자료 자체가 에듀플로 결과물입니다. 아래 요소들이 어디에 있는지 찾아보세요.

Step 2: 나의 교재 주제 한 줄 적기 (10분)

섹션 제목: “Step 2: 나의 교재 주제 한 줄 적기 (10분)”

이 활동의 의도: 2차시부터 실습에 쓸 ‘나의 주제’를 미리 정해두면, 실습 시간이 훨씬 풍성해집니다.

활동지: 내 교과에서 만들고 싶은 교재 한 줄 기획서

아래 문장을 채워보세요. 완벽할 필요 없습니다. 2차시에서 수정할 수 있습니다.

“저는 _____학년 _____과 학생들을 위해, _____에 대한 교재를 만들고 싶습니다. 이 교재를 읽고 나면 학생들이 _____를 할 수 있으면 좋겠습니다.”

작성 예시 (정보 교과):

“저는 중학교 2학년 정보 학생들을 위해, 스크래치로 게임 만들기 교재를 만들고 싶습니다. 이 교재를 읽고 나면 학생들이 간단한 피하기 게임을 스스로 설계·구현할 수 있으면 좋겠습니다.”

작성 예시 (일반 교과):

“저는 고등학교 1학년 국어 학생들을 위해, 현대시 감상법 교재를 만들고 싶습니다. 이 교재를 읽고 나면 학생들이 시의 화자·정서·주제를 스스로 분석할 수 있으면 좋겠습니다.”

Step 3: 2차시 준비물 체크 (5분)

섹션 제목: “Step 3: 2차시 준비물 체크 (5분)”

💡 관리자 승인이 뭔가요? 에듀플로는 교사 대상 플랫폼이라, 가입 후 관리자가 교사 여부를 확인하고 승인해 드립니다. 연수 담당 선생님께 미리 Gmail 주소를 알려두시면, 2차시 시작 시점에 이미 승인되어 있습니다.

⚠️ 주의할 점

섹션 제목: “⚠️ 주의할 점”

1. 에듀플로는 “교재용” 도구입니다. 시험지 출제, 학생 개별 피드백, 학부모 알림 같은 업무는 지원하지 않습니다. “수업에서 함께 읽을 자료”를 만드는 데 특화되어 있습니다.

2. Step 1(방향 논의)을 대충 넘기지 마세요. “누구에게, 왜, 어떤 톤으로”를 흐릿하게 답하면 최종 교재도 흐릿하게 나옵니다. 빵 주문할 때 “그냥 빵 주세요”라고 하면 원하는 빵이 안 나오는 것과 같습니다. Step 1에서 5분 더 쓰면 Step 4 결과물이 2배 좋아집니다.

✅ 점검하기

섹션 제목: “✅ 점검하기”

1. 에듀플로를 한 문장으로 설명한다면?

정답 확인

“선생님이 교재의 방향을 대화로 알려주면, AI가 목차부터 웹사이트 배포까지 자동으로 만들어 주는 교육자료 생성 플랫폼”

핵심 키워드: ① 대화 기반 ② 목차→본문→배포 자동화 ③ 교재 특화

2. 다음 중 에듀플로의 5단계 순서로 올바른 것은?

① 목차 제안 → 방향 논의 → 챕터 생성 → 목차 확정 → 배포 ② 방향 논의 → 목차 제안 → 목차 확정 → 챕터 생성 → 배포 ③ 방향 논의 → 챕터 생성 → 목차 제안 → 목차 확정 → 배포 ④ 목차 확정 → 방향 논의 → 목차 제안 → 챕터 생성 → 배포

정답 확인

② 방향 논의 → 목차 제안 → 목차 확정 → 챕터 생성 → 배포

“설계(1-3) → 제작(4) → 납품(5)“의 흐름입니다. 설계가 먼저, 제작이 다음, 배포가 마지막입니다.

3. ChatGPT로 교재를 만들 때 에듀플로 대비 불편한 점 세 가지는?

정답 확인

① 기억 소실 — 대화가 길어지면 앞 챕터 내용·용어를 잊음 ② 템플릿 불일치 — 매 챕터 형식이 미묘하게 달라져 직접 손봐야 함 ③ 배포 공백 — 텍스트만 나와서, 학생 배포까지 수동 작업 필요

에듀플로는 이 세 가지를 각각 컨텍스트 고정, 템플릿 강제, 웹사이트 자동 배포로 해결합니다.

4. (서술형) 선생님의 교과에서 에듀플로로 만들고 싶은 교재를 한 문장으로 적어보세요.

  • 도달 목표: 읽고 나면 뭘 할 수 있어야? (예: 피하기 게임 구현)

이 문장이 2차시 실습의 출발점이 됩니다. 너무 넓으면(예: “프로그래밍 전체”) AI가 방향을 못 잡고, 너무 좁으면(예: “if문 한 개 쓰기”) 한 챕터도 안 나옵니다. “10차시 수업 분량” 정도가 적당합니다.
